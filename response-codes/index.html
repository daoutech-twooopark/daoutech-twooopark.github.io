응답 코드

서비스별 결과 코드입니다. 응답 본문의 code 필드가 처리 결과를 나타내며(1000=성공), 발송 후 통신사·카카오·RCS·네이버가 반환하는 채널별 전송 결과 코드는 각 서비스 탭의 품목별 표를 참고하세요.

HTTP

비즈뿌리오 API는 인증 오류(4xx)와 Rate Limit(429)를 제외하면 HTTP 상태를 따로 명시하지 않고 200으로 응답합니다. 실제 성공/실패는 응답 본문의 code 값으로 판단하세요.
